Privacy Policy

Product Summarizer & Review Analyzer

This extension extracts publicly available product descriptions and customer reviews from supported e-commerce product pages (Amazon, Lazada, Shopee) only when the user activates the extension.

Data Collection

The extension does not collect, store, or share personal information.

Processing

Product information is processed solely to generate summaries and sentiment analysis. No browsing history is tracked, and no data is sold or shared with third parties.

AI Processing

If AI processing is used, only the product description and review text selected by the user is sent to the processing API. No personal account information is collected.

Local Storage

All user preferences are stored locally using Chrome storage and are not transmitted externally.
